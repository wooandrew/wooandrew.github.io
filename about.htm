What I Use:

  • Laptop | HP-15-ay039wm

    • Self explanatory. This is what I use on a daily basis for pretty much everything.

  • Code::Blocks

    • Code::Blocks is a minimalistic C++ IDE. It has less features than Visual Studio, but is also less convoluted, which is why it is my preferred C++ IDE.
    http://www.codeblocks.org/

  • PyCharm

    • PyCharm is one of the best, if not THE best, Python IDE. With a simplistic and friendly UI, it is definitely one of my favourites of all time.
    https://www.jetbrains.com/pycharm/

  • Adobe Brackets

    • While not my first choice, the fact that Adobe Brackets is one of the only HTML/CSS Editors with live offline development makes it my preferred tool for web development.
    http://brackets.io/

  • Firefox Developer Edition

    • The absolute best browser ever. Built specifically for web designers and first of its kind, Firefox Developer Edition in conjunction with Adobe Brackets is irreplaceable.
    https://www.mozilla.org/en-US/firefox/developer/

  • Eclipse Oxygen

    • I hate Java. Nonetheless, I feel it is important to understand the concepts of such a popular language, so whenever I code with Java, I use Eclipse Oxygen.
    https://www.eclipse.org/

  • RAPTOR by USAF

    • RAPTOR holds a special place in my heart: It taught me how to code. Developed by the U.S Airforce, RAPTOR helps people visualize code.
    http://raptor.martincarlisle.com/

  • MIT Scratch

    • I don't do much with MIT Scratch. But it's pretty cool, and I use it to teach programming.
    https://scratch.mit.edu/

  • Blender on Steam

    • Blender is a 3D Modeling program. I use the Steam version for the automatic updates. The logo for Encrypter was designed using Blender.
    https://www.blender.org/

  • Inkscape

    • Inskape is a great free replacement for Photoshop. It has similar features, and is simple to use. The logo for Directory.h was designed using it.
    https://inkscape.org/en/
