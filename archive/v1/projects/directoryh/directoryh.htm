  • Directory.h is a C++ Header file that allows users to easily import the path of the current running program.

  • This header file is created for educational purposes only.

  • How to use?

    1. Download the header file here and save it to your project folder. It should look something like this:
    2. Add #include "directory.h" at the way top with all of your other includes.


    3. Next, set string path as dir.getDir(), like so: string path = dir.getDir();


    4. Finally, use path on whatever you want. In this case, my program returns the full working directory.
